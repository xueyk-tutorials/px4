# VTOL

## 接线

| Port   | Connection                        |
| ------ | --------------------------------- |
| MAIN 1 | 右前电机，Front right motor (CCW) |
| MAIN 2 | 左后电机，Back left motor (CCW)   |
| MAIN 3 | 左前电机，Front left motor (CW)   |
| MAIN 4 | 右后电机，Back right motor (CW)   |
| AUX 1  | 左副翼，Left aileron              |
| AUX 2  | 右副翼，Right aileron             |
| AUX 3  | 升降，Elevator                    |
| AUX 4  | 方向，Rudder                      |
| AUX 5  | 前拉/尾推电机，Throttle           |
